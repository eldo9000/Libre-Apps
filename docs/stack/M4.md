# Stack M4 — Pro Features

**Phase:** 3 (Pro Tiers & Polish)
**Status:** Not started
**Depends on:** M3

---

## Goal

Pro tier for writers and professionals who live in their editor. $3–5 one-time.

---

## Pro Features

### Version History
- [ ] Auto-save snapshots every N minutes (configurable)
- [ ] Version timeline showing snapshots with timestamps
- [ ] Diff view between any two versions
- [ ] Restore to any previous version
- [ ] Snapshots stored in SQLite alongside the file (`.stack-history/`)

### Export Formats
- [ ] Export to PDF (styled, with page breaks at H1)
- [ ] Export to DOCX (via pandoc or custom Rust implementation)
- [ ] Export templates (academic paper, blog post, documentation)
- [ ] Custom CSS for export styling

### Templates
- [ ] New-from-template: blog post, meeting notes, project README, journal entry
- [ ] Custom template creation (save any document as template)
- [ ] Template variables (`{{date}}`, `{{title}}`) auto-filled on creation

### Sync
- [ ] Watch folder: auto-open files from a synced directory
- [ ] Conflict resolution if file changed externally while editing
- [ ] Optional git integration (auto-commit on save to a configured repo)

---

## Free vs Pro Boundary

**Free:** Editor, multi-tab, outline, search/replace, format controls, syntax highlighting, markdown shortcuts, export to HTML/TXT/PDF (basic), live preview, spell check.

**Pro:** Version history with diff, DOCX export, templates, folder sync, git integration.

The free version is a complete, excellent markdown editor. Pro is for people who write professionally and need history, templates, and export flexibility.

# Stack M3 — Live Preview & Spell Check

**Phase:** 2 (Power Features & Integration)
**Status:** Not started
**Depends on:** M2 (markdown experience)

---

## Goal

Stack becomes a complete markdown writing environment with real-time preview and spell checking.

---

## Tasks

### Live Preview

- [ ] Toggle button: Editor only | Split view (editor + preview) | Preview only
- [ ] Preview renders markdown to styled HTML (reuse export_html engine from pulldown-cmark)
- [ ] Preview scrolls in sync with editor cursor position
- [ ] Preview updates on every keystroke (debounced ~150ms)
- [ ] Preview respects current theme (dark/light)
- [ ] Preview uses same Geist font and spacing as the editor
- [ ] Links in preview are clickable (open in default browser via xdg-open)
- [ ] Images in preview load from relative file paths
- [ ] Code blocks in preview have syntax highlighting (highlight.js or Shiki, lightweight)

### Spell Check

- [ ] Integrate system spell checker (hunspell via Rust or WebKit built-in)
- [ ] Red wavy underline on misspelled words
- [ ] Right-click context menu with spelling suggestions
- [ ] "Add to Dictionary" option
- [ ] Language auto-detection or manual selection
- [ ] Spell check toggle (on/off) in Format panel
- [ ] Skip code blocks, URLs, and file paths

### Tab Drag-to-Break

- [ ] Drag a tab out of the tab bar → creates a new Stack window with that tab
- [ ] New window inherits theme/accent settings
- [ ] "Merge Windows" option to consolidate tabs back

### Tests

- [ ] Test preview renders headings, lists, code blocks, images, links correctly
- [ ] Test scroll sync between editor and preview
- [ ] Test spell check ignores code blocks

---

## Definition of Done

- Split view shows real-time markdown preview alongside editor
- Spell check catches common misspellings with suggestions
- Tabs can be dragged out to create new windows

## Note on Milestone Numbering

Prism has one fewer Phase 1 milestone than Shelf, Stack, and Fade (those apps have two Phase 1 milestones each; Prism has one). This shifts Prism's milestones relative to the other apps:

| Phase | Shelf/Stack/Fade | Prism |
|-------|-----------------|-------|
| Phase 1 | M1 + M2 | M1 |
| Phase 2 | M3 | M2 |
| Phase 3 (Pro) | M4 | M3 |

**Prism's pro tier is documented in [M3.md](M3.md).** It covers annotation, RAW adjustments, and advanced slideshow. There is no additional M4 scope — Prism is complete at M3.

---

## Pro Tier Summary (see M3.md for full spec)

The pro features that ship at M3:

- **Annotation** — draw on images, color picker, non-destructive save, annotation layers
- **RAW Adjustments** — exposure, white balance, contrast, saturation, histogram, before/after comparison, sidecar files
- **Advanced Slideshow** — multiple transition types, per-image duration, background music, export to MP4

---

## Future Work (If Any)

If Prism ever gains scope beyond what M3 covers, it should be written as a new milestone with a concrete spec — not retrofitted into this document. The absence of M4 scope today reflects that M3 is a complete, well-scoped pro tier.

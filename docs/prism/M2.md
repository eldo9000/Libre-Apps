# Prism M2 — Slideshow & Playlist

**Phase:** 2 (Power Features & Integration)
**Status:** Not started
**Depends on:** M1 (metadata & UX polish)

---

## Goal

Prism handles collections of files, not just single files. Slideshow for images, playlist for media.

---

## Tasks

### Slideshow Mode

- [ ] Enter slideshow from any image (F5 or button)
- [ ] Full-screen display with auto-advance
- [ ] Configurable duration per slide (3s, 5s, 10s, 15s, 30s, custom)
- [ ] Transition: crossfade (simple, smooth)
- [ ] Controls overlay (pause/play, next/prev, duration, exit) — auto-hide after 3s
- [ ] Manual navigation with arrow keys during slideshow
- [ ] Shuffle mode
- [ ] Loop toggle
- [ ] Slideshow uses all images in current directory (or selected subset)
- [ ] Escape exits slideshow

### Playlist / Queue

- [ ] Queue panel (sidebar) showing upcoming files
- [ ] Add files to queue via drag-and-drop or "Add to Queue" context menu
- [ ] Reorder queue by drag
- [ ] Remove individual items from queue
- [ ] Auto-advance to next item when current finishes (video/audio)
- [ ] Repeat modes: none, repeat one, repeat all
- [ ] Queue persists during session (not across app restarts)

### Batch Export

- [ ] Select multiple files in queue → "Export All" option
- [ ] Export images to different format (e.g., all PNGs → JPEGs)
- [ ] Export with resize (maintain aspect ratio)
- [ ] Progress indicator for batch operations
- [ ] Output to specified directory

---

## Definition of Done

- Slideshow mode works for image directories with configurable timing
- Media files can be queued and play sequentially
- Batch export converts multiple files with progress feedback

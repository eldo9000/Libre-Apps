**Phase:** 1 (Foundation & MVP Completion)

## Goal

Prism's core viewing works well. This milestone adds the features users expect from a media viewer: seeing file details, opening files easily, and knowing what shortcuts are available.

## Implementation Notes

- **EXIF**: crate `kamadak-exif` (module name `exif`) for images. GPS formatted as `{decimal}° N/S/E/W`.
- **Video/Audio**: ffprobe subprocess with `-print_format json -show_streams -show_format`. If ffprobe not in PATH, `codec` field surfaces the error string; no panic.
- **PDF**: `lopdf` pure-Rust crate. PDF strings decoded as UTF-16BE (BOM check) or raw UTF-8.
- **3D**: OBJ counts `v `/`f `/`mtllib ` lines; STL detects ASCII vs binary; GLTF/GLB parses JSON accessor structure.
- **Image dimensions**: header-parsed (PNG IHDR, GIF header, WebP VP8, JPEG SOF scan, BMP DIB header); EXIF PixelXDimension/PixelYDimension override when present.
- **Shortcut data** extracted to `src/lib/shortcuts.js` (plain JS module) so tests import it directly without mounting the Svelte component.
- **`dragDropEnabled`** not a valid Tauri 2 `app` config field — HTML5 drag-drop events pass through the WebView by default.

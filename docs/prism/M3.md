# Prism M3 — Pro Features

**Phase:** 3 (Pro Tiers & Polish)
**Status:** Not started
**Depends on:** M2

---

## Goal

Pro tier for photographers, designers, and media professionals. $3–5 one-time.

---

## Pro Features

### Annotation
- [ ] Draw on images (freehand, arrows, rectangles, circles, text)
- [ ] Color picker for annotation tools
- [ ] Save annotated version (non-destructive — original preserved)
- [ ] Annotation layers (toggle visibility)

### RAW Adjustments
- [ ] Exposure, white balance, contrast, saturation sliders for RAW images
- [ ] Histogram display
- [ ] Before/after comparison (split view or toggle)
- [ ] Non-destructive: adjustments saved as sidecar file

### Advanced Slideshow
- [ ] Multiple transition types (crossfade, slide, zoom, ken burns)
- [ ] Per-image duration override
- [ ] Background music (select audio file)
- [ ] Export slideshow as video (MP4)

---

## Free vs Pro Boundary

**Free:** View all formats, zoom, rotate, keyboard shortcuts, metadata viewer, slideshow (basic), playlist, file navigation, drag-and-drop.

**Pro:** Annotation tools, RAW adjustments, advanced slideshow with export, batch export with custom settings.

The free version is a complete, excellent media viewer. Pro is for people who need to annotate, adjust RAW files, or create presentations.

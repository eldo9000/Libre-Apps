# Fade M4 — AI Features Phase 2 (Pro)

**Phase:** 3 (Pro Tiers & Polish)
**Status:** Not started
**Depends on:** M3

---

## Goal

Advanced AI features that justify a pro tier. $3–5 one-time. These are computationally expensive and niche — power users who need them will pay gladly.

---

## Pro Features

### Stem Separation (Audio)
- [ ] Integrate Demucs (Meta's open-source stem separator)
- [ ] Separate into: vocals, drums, bass, other
- [ ] Output individual stems as WAV/FLAC files
- [ ] Preview: solo/mute individual stems before export
- [ ] Use case: music production, karaoke, remixing

### Face Restoration (Image)
- [ ] Integrate GFPGAN
- [ ] Restore old/damaged/low-res faces in photos
- [ ] Before/after preview
- [ ] Batch processing for photo collections
- [ ] Use case: restoring family photos, improving old scans

### Frame Interpolation (Video)
- [ ] Integrate RIFE (Real-time Intermediate Flow Estimation)
- [ ] Convert 30fps → 60fps or 24fps → 60fps
- [ ] Useful for old footage, slow-motion creation
- [ ] Progress reporting (frame-by-frame)

### Background Removal (Image)
- [ ] Integrate rembg (U²-Net)
- [ ] One-click background removal
- [ ] Output as PNG with transparency
- [ ] Batch processing support
- [ ] Preview with checkerboard background

### Video Stabilization
- [ ] Use FFmpeg's vidstabtransform (two-pass)
- [ ] Or integrate Gyroflow for gyroscope-based stabilization
- [ ] Before/after preview (side-by-side frames)
- [ ] Crop compensation slider (more stable = more crop)

### Color Grading
- [ ] LUT (Look-Up Table) application to video/images
- [ ] Ship a small set of default LUTs (cinematic, vintage, B&W, warm, cool)
- [ ] Import custom .cube LUT files
- [ ] Preview LUT effect on a single frame before applying

---

## Free vs Pro Boundary

**Free:** Format conversion (image/video/audio), batch queue, presets, resize, crop, rotate, trim, audio extraction, loudness normalization, noise removal (Phase 1), upscaling (Phase 1), auto-captions (Phase 1).

**Pro:** Stem separation, face restoration, frame interpolation, background removal, video stabilization, LUT color grading.

The free version handles all standard media conversion plus basic AI. Pro is for creative professionals who need production-level AI processing.

# Splice M3 — AI Features Phase 1

**Phase:** 2 (Power Features & Integration)
**Status:** Not started
**Depends on:** M2 (preview & parallel encoding)

---

## Goal

First wave of AI-powered features. Focus on the most universally useful ones that run locally without a GPU requirement.

---

## Tasks

### Noise Removal (Audio)
- [ ] Integrate RNNoise or DeepFilterNet (lightweight, CPU-capable)
- [ ] One-click "Remove Background Noise" toggle in audio options
- [ ] Before/after preview (play 3-second sample with and without noise removal)
- [ ] Works on voice recordings, podcasts, interviews
- [ ] Chain with existing audio conversion (denoise → convert in one pass)

### Image Upscaling
- [ ] Integrate Real-ESRGAN (ncnn-vulkan variant for GPU-optional)
- [ ] Upscale modes: 2x, 4x
- [ ] Model options: general (realesrgan-x4plus), anime (realesrgan-x4plus-anime)
- [ ] Falls back to CPU if no Vulkan GPU detected
- [ ] Progress reporting (can be slow — set expectations in UI)
- [ ] Preview: show small crop of upscaled result before full processing

### Auto-Captions (Video)
- [ ] Integrate whisper.cpp (CPU inference, no Python/CUDA required)
- [ ] "Generate Subtitles" button in video options
- [ ] Output: SRT file alongside the video
- [ ] Optional: burn subtitles into video (FFmpeg subtitle filter)
- [ ] Language auto-detection
- [ ] Model sizes: tiny (fast, less accurate) and base (slower, more accurate)
- [ ] Progress: show transcription progress as percentage

### Shared AI Infrastructure
- [ ] Model download manager: first-use downloads model files to `~/.local/share/librewin/models/`
- [ ] Download progress shown in UI
- [ ] Model files are shared across apps (Splice + future Pilot integration)
- [ ] Health check: verify model files are intact before use
- [ ] Graceful fallback: if model is missing/corrupted, offer to re-download (don't crash)

### Tests
- [ ] Test noise removal on sample audio file (verify output is different from input)
- [ ] Test upscaling on small test image (verify output dimensions)
- [ ] Test caption generation on short test clip (verify SRT output format)
- [ ] Test model download/verification flow

---

## Definition of Done

- Audio noise removal works on voice recordings with audible improvement
- Image upscaling produces higher-resolution output with visible quality improvement
- Video captions generate accurate SRT files for English speech
- AI models download on first use with progress indication
- All AI features work on CPU (GPU is optional performance bonus)

# Shelf M3 — Power Features

**Phase:** 2 (Power Features & Integration)
**Status:** Not started
**Depends on:** M2 (core polish)

---

## Goal

Shelf becomes a power-user file manager. Global access, system integration, Wine bridge.

---

## Tasks

### Shelf Mode (Floating Palette)

- [ ] Global shortcut (Super+E already exists) toggles between full window and compact floating mode
- [ ] Compact mode: 400×500 floating window, always-on-top, single-pane
- [ ] Quick search focus on summon — type immediately to find files
- [ ] Escape dismisses floating mode
- [ ] Recent files list in floating mode for fast access

### xdg-desktop-portal Backend

- [ ] Register Shelf as a system file picker via xdg-desktop-portal
- [ ] Other apps' "Open File" / "Save File" dialogs use Shelf's UI
- [ ] Support `org.freedesktop.portal.FileChooser` interface
- [ ] Handle multiple selection mode
- [ ] Handle directory selection mode

### Wine Drag-Bridge

- [ ] Detect active Wine windows via `xdotool`
- [ ] When dragging a file from Shelf toward a Wine window, inject the Windows path via `winepath`
- [ ] Use xdotool to simulate the path paste into Wine's file dialog
- [ ] Only activate when Wine is detected (get_wine_status already exists)

### Filesystem Indexing

- [ ] Background index using `plocate` or custom SQLite FTS
- [ ] Instant cross-filesystem search (not just current directory filter)
- [ ] Index updates on file change events (inotify/fanotify)
- [ ] Search results show file path, size, modified date, and matched context

---

## Definition of Done

- Shelf Mode summons a floating mini file browser from any workspace
- System file dialogs (in GTK/Qt apps) can optionally use Shelf
- Files can be dragged from Shelf into Wine application dialogs
- Cross-filesystem search returns results in under 500ms

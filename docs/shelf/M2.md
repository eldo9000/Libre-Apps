# Shelf M2 — Core Polish

**Phase:** 1 (Foundation & MVP Completion)
**Status:** Not started
**Depends on:** M1 (archive integration)

---

## Goal

Make Shelf reliable and complete enough to be someone's daily file manager. Fix the rough edges, add missing basics, improve error handling.

---

## Tasks

### Batch Operations

- [ ] Multi-select: Ctrl+Click for individual, Shift+Click for range
- [ ] Selected items counter in status bar
- [ ] Context menu for multi-select: Copy, Move, Delete, Tag All
- [ ] Copy/Move with progress dialog (for large operations)
- [ ] Delete with confirmation dialog (show file count)
- [ ] Drag-and-drop to move files between panes (dual-pane mode)

### File Previews

- [ ] Thumbnail generation for images (lazy-load, cache in `/tmp/shelf-thumbs/`)
- [ ] Toggle between list view and grid/thumbnail view
- [ ] Preview panel (optional third column) showing selected file details + thumbnail

### Hot-Plug Volume Detection

- [ ] Replace startup-only volume scan with filesystem watcher (inotify on `/dev/disk/`)
- [ ] New volumes appear in sidebar automatically
- [ ] Removed volumes disappear + show toast if user was browsing that volume
- [ ] USB drive eject/unmount option in context menu

### Error Handling

- [ ] Failed file operations show error dialog (not just toast)
- [ ] Dual-pane refresh: file changes in pane A trigger refresh in pane B if same directory
- [ ] Stale directory detection: if current path is deleted externally, show "folder moved/deleted" state
- [ ] Permission errors: clear message when trying to access restricted directories

### Keyboard Navigation

- [ ] Vi-key optional mode (j/k for up/down, l to enter, h to go back)
- [ ] Ctrl+L to focus address bar / path input
- [ ] Tab to cycle between sidebar, file list, and pane B
- [ ] Delete key triggers delete confirmation

### Tests

- [ ] Frontend: vitest smoke tests for navigation, tab management, search filtering
- [ ] Rust: unit tests for file listing, tag read/write, volume parsing
- [ ] Integration: test batch copy/move/delete operations

---

## Definition of Done

- Multi-select and batch operations work reliably
- Thumbnails load for image files in the listing
- USB drives appear/disappear without app restart
- Error states are handled with user-friendly messages
- Core keyboard shortcuts documented and functional
- Test suite covers critical paths

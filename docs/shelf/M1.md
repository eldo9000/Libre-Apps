# Shelf M1 — Archive Manager Integration

**Phase:** 1 (Foundation & MVP Completion)
**Status:** Not started
**Depends on:** Shared CI setup

---

## Goal

Archive browsing and extraction is no longer a separate app — it's a built-in Shelf feature. Users should be able to browse into archives like folders and extract files without leaving the file manager.

---

## Tasks

### Rust Backend

- [ ] Add `zip` crate (or `zip2`) for ZIP read/write support
- [ ] Add `sevenz-rust` or shell out to `7z` for 7Z support
- [ ] Add `tar` + `flate2` crates for TAR/GZ support
- [ ] New command: `list_archive(path)` → returns Vec<ArchiveEntry> with name, size, modified date, is_dir
- [ ] New command: `extract_archive(path, dest)` → extracts full archive to destination
- [ ] New command: `extract_files(path, files, dest)` → extracts selected files only
- [ ] New command: `archive_info(path)` → returns format, total size, file count, compressed size
- [ ] Handle nested archives (archive within archive) — display, don't auto-extract

### Frontend (QuickFiles.svelte)

- [ ] Detect archive files by extension (.zip, .7z, .tar, .tar.gz, .tar.bz2, .tar.xz, .rar)
- [ ] Double-click/Enter on archive → browse into it (call `list_archive` instead of `list_dir`)
- [ ] Breadcrumb trail shows archive path distinctly (e.g., icon or color change for archive segment)
- [ ] "Extract Here" and "Extract To..." in context menu for archives
- [ ] "Extract Selected" when inside an archive with files selected
- [ ] Back button exits archive and returns to parent directory
- [ ] Archive files show compressed/uncompressed size in file list

### Edge Cases

- [ ] Password-protected ZIPs — prompt for password, display error if wrong
- [ ] Corrupted archives — show error toast, don't crash
- [ ] Very large archives (10,000+ files) — paginate or lazy-load the listing
- [ ] Symlinks inside archives — display but warn about extraction outside target

### Tests

- [ ] Rust unit tests for archive listing (ZIP, 7Z, TAR)
- [ ] Rust unit tests for extraction (full + selective)
- [ ] Test corrupted archive handling
- [ ] Test nested archive detection

---

## Definition of Done

- User can double-click a ZIP/7Z/TAR file and browse its contents like a folder
- User can extract full archive or selected files via context menu
- Archives are visually distinct in the breadcrumb trail
- Corrupted/password-protected archives show appropriate errors
- Rust tests pass for all archive operations

# Shelf M4 — Pro Features

**Phase:** 3 (Pro Tiers & Polish)
**Status:** Not started
**Depends on:** M3

---

## Goal

Pro tier for power users who need more from their file manager. $3–5 one-time.

---

## Pro Features

### Network Drives
- [ ] SMB/CIFS share browsing via `smbclient` or `gio`
- [ ] SFTP/SSH browsing via Rust SSH crate
- [ ] Saved connection profiles
- [ ] Auto-reconnect on network change

### Advanced Batch Operations
- [ ] Batch rename with pattern (regex, sequential numbering, date prefix)
- [ ] Find & replace in filenames
- [ ] Batch tag assignment across multiple directories
- [ ] Duplicate file finder (hash-based)

### Sync
- [ ] Two-way folder sync between local directories
- [ ] Sync profiles (source → destination, schedule, conflict resolution)
- [ ] Dry-run mode showing what would change before executing

### Plugin System
- [ ] Lua or WASM-based plugin API
- [ ] Custom context menu items via plugins
- [ ] Custom sidebar panels via plugins
- [ ] Plugin marketplace (curated, not open)

---

## Free vs Pro Boundary

**Free:** File management, tags, dual pane, tabs, search, archive browsing, quick convert, keyboard navigation, Shelf Mode, system file picker integration.

**Pro:** Network drives, advanced batch operations, folder sync, plugin system.

The free version is a complete, excellent file manager. Pro is for people who manage complex file workflows across networks.

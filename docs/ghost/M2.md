# Ghost M2 — Settings & Browser Fundamentals

**Phase:** 1 (Foundation & MVP Completion)
**Status:** Not started
**Depends on:** M1 (privacy features)

---

## Goal

Ghost needs basic browser features that are currently missing: settings UI, zoom, fullscreen, find-in-page. Without these, it doesn't function as a usable browser.

---

## Tasks

### Settings UI

- [ ] Settings panel (slide-out from right or separate page)
- [ ] **Privacy section:**
  - Toggle: Block third-party cookies (default: on)
  - Toggle: Block trackers (default: on)
  - Toggle: Disable WebRTC (default: on)
  - Toggle: HTTPS-only mode (default: off)
  - Toggle: Enable JavaScript (default: on)
- [ ] **Appearance section:**
  - Toggle: Dark mode for web content (WebKit dark reader)
  - Font size override for web content
- [ ] **Search section:**
  - Default search engine selection (DuckDuckGo, Google, Brave, Startpage, Kagi)
  - Custom search engine URL pattern
- [ ] Settings persist to `~/.config/librewin/ghost-settings.json`
- [ ] Settings load on app start

### Zoom Controls

- [ ] `Ctrl+=` zoom in, `Ctrl+-` zoom out, `Ctrl+0` reset to 100%
- [ ] Zoom level shown in toolbar (e.g., "125%")
- [ ] Pinch-to-zoom on trackpad
- [ ] Zoom persists per-site during session (resets on close)

### Fullscreen

- [ ] `F11` toggles fullscreen
- [ ] Toolbar auto-hides in fullscreen (move mouse to top to reveal)
- [ ] Escape exits fullscreen

### Find in Page

- [ ] `Ctrl+F` opens find bar
- [ ] Highlight all matches on page
- [ ] Next/Previous match navigation (Enter / Shift+Enter)
- [ ] Match count display (e.g., "3 of 12")
- [ ] Escape closes find bar

### Tab Support (Basic)

- [ ] `Ctrl+T` new tab
- [ ] `Ctrl+W` close tab
- [ ] `Ctrl+Tab` / `Ctrl+Shift+Tab` switch tabs
- [ ] Tab bar below titlebar showing page titles
- [ ] Each tab is an independent webview
- [ ] New tab opens the Ghost homepage (search links + shortcuts)

### Tests

- [ ] Test settings persist and load correctly
- [ ] Test zoom level changes
- [ ] Test find-in-page highlights matches

---

## Definition of Done

- Settings UI allows configuring all privacy toggles
- Settings persist across app restarts
- Zoom, fullscreen, and find-in-page all work
- Basic tab support (open, close, switch)
